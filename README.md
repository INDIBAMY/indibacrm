# INDIBA CRM — IONiaga Malaysia

Customer relationship management system for IONiaga Sdn Bhd.

## Setup

1. Run `supabase_setup.sql` in your Supabase SQL Editor
2. Upload `index.html` to this GitHub repository
3. Enable GitHub Pages: Settings → Pages → Source: main branch
4. Access at: `https://yourusername.github.io/indiba-crm`
